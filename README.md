#### "Brain Games" project with 5 different math games!

##### Requirements:

1. python = ^3.10
2. prompt = ^0.4.1

##### Installation:

1. Clone the project

```commandline
git clone https://github.com/tanuki-evil1/python-project-49.git
```

2. Go to the project directory

```commandline
cd python-project-49
```

3. Install dependencies

```commandline
make install
```

4. Build the project

```commandline
make build
```

5. Install package

```commandline
make package-install
```
